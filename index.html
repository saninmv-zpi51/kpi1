Тема власного WEB-застосунку

WEB-застосунок «Exhibitum» для інформування користувачів про музейні виставки, події та експозиції музеїв України.

Мета створення WEB-застосунку

Забезпечити зручний, швидкий та візуально зрозумілий доступ до актуальної інформації про музейні виставки, їх тематику, дати проведення, місце проведення та короткий опис, а також сприяти популяризації культурної спадщини.

Скорочений опис предметного середовища (бізнес-логіка)

WEB-застосунок «Exhibitum» надає користувачам інформацію про поточні та майбутні музейні виставки і події. Користувач може переглядати перелік виставок, отримувати детальну інформацію про кожну експозицію, дізнаватись про місце проведення, дати та тематику заходу. Система передбачає можливість адміністрування контенту, зокрема додавання, редагування та видалення інформації про виставки та музеї.

Функціональні вимоги

  • перегляд актуальних та майбутніх музейних виставок;
  • перехід між основними розділами веб-застосунку;
  • пошук та фільтрація інформації;
  • відображення партнерських музеїв та тематичних матеріалів.

Нефункціональні вимоги

  • зручний та інтуїтивно зрозумілий інтерфейс;
  • адаптивність під різні розміри екранів;
  • швидке завантаження сторінок;
  • коректна семантична HTML-розмітка та кросбраузерність (Chrome, Firefox, Edge).

Структура WEB-застосунку побудована з використанням семантичних HTML5-тегів <header>, <nav>, <main>, <section>, <article> та <footer>.

У звітному HTML-документі додатково наведено приклади використання таблиць, зображень, маркованих списків та елементів форм.

HTML-код таблиці (інформація про виставки)

<table>
              <caption>Актуальні музейні виставки</caption>

              <thead>
                <tr>
                  <th>Назва виставки</th>
                  <th>Музей</th>
                  <th>Дати проведення</th>
                </tr>
              </thead>

              <tbody>
                <tr>
                  <td>Мистецтво ХХ століття</td>
                  <td>Музей Ханенків</td>
                  <td>01.03 – 30.04</td>
                </tr>

                <tr>
                  <td>Український авангард</td>
                  <td>NAMU</td>
                  <td>15.04 – 20.06</td>
                </tr>
              </tbody>
            </table>

Теорія (коротко)

  • <table> — контейнер для табличних даних.
  • <tr> — рядок таблиці.
  • <th> — заголовкова комірка.
  • <td> — комірка з даними.
  • <caption> — підпис таблиці.
  • <thead> та <tbody> — логічне розділення заголовку і вмісту.

HTML-код зображення

<img src="images/homepage2.png" alt="Головна сторінка Exhibitum" style="max-width:100%;">

Теорія (атрибути)

  • src — шлях до зображення.
  • alt — альтернативний текст (доступність).
  • width/height — розміри (краще керувати через CSS).

HTML-код списків (використання у WEB-застосунку «Exhibitum»)

<!-- Маркірований список: можливості користувача -->
              <ul>
                <li>Перегляд актуальних музейних виставок</li>
                <li>Ознайомлення з описом експозицій</li>
                <li>Навігація між розділами сайту</li>
              </ul>

              <!-- Нумерований список: дії користувача -->
              <ol>
                <li>Вибір музею або виставки</li>
                <li>Перегляд детальної інформації</li>
                <li>Ознайомлення з датами та місцем проведення</li>
              </ol>

Теорія

  • <ul> — маркірований список.
  • <ol> — нумерований список.
  • <li> — елемент списку.

Способи підключення стилів

1) Зовнішній CSS-файл (використано у «Exhibitum»)

<link rel="stylesheet" href="style.css">

2) Внутрішні стилі (приклад)

<style>
body{ font-family: Arial, sans-serif; }
</style>

3) Інлайн-стиль (приклад)

<h1 style="color:#00595C;">Exhibitum</h1>

У проєкті «Exhibitum» основним способом є підключення зовнішнього CSS-файлу, оскільки це забезпечує централізоване керування стилями та зручність підтримки.

Селектори тегу

HTML (фрагмент)

<body>
  <header class="header">...</header>
  <main>...</main>
</body>

CSS (фрагмент з проєкту)

body{
  margin: 0;
  background: var(--white);
  color: var(--text);
  font-family: Inter, system-ui, -apple-system, Segoe UI, Roboto, Arial, sans-serif;
}

Селектори класу

HTML (фрагмент)

<div id="museumStrip">
  <class="museum-strip">
  </class>
</div>

CSS (фрагмент з проєкту)

#museumStrip{
  display: flex;
  gap: 28px;
  transition: transform 1.2s ease-in-out;
}

Селектори ідентифікаторів

HTML (фрагмент)

<div class="museum__bar--carousel">
  <div id="museumStrip" class="museum-strip">
    <img class="museum-strip__logo" src="images/logo-ivan-honchar.webp" alt="...">
    ...
  </div>
</div>

CSS (приклад використання #id)

#museumStrip{
  display: flex;
  align-items: center;
  gap: 28px;
  height: 100%;
  transition: transform 1.2s ease-in-out;
  will-change: transform;
}

Ідентифікатор використовується для звернення до конкретного елемента сторінки. У проєкті #museumStrip застосовується для керування горизонтальною смужкою логотипів.

Інші селектори

Універсальний селектор *

*{ box-sizing: border-box; }

Дочірній селектор (A B)

.header__logo img{
  height: 100%;
  width: auto;
  display: block;
  object-fit: contain;
}

Сусідній селектор (A + B) — приклад

h2 + p{
  margin-top: 6px;
}

Селектор атрибута — приклад

a[aria-label]{
  text-decoration: none;
}

Універсальний селектор застосовується для базового налаштування, дочірній — для стилізації елементів всередині блоків, селектор сусіднього елемента — для керування відступами, селектор атрибута — для точкової стилізації елементів за атрибутами.

CSS: Шрифти Текст Таблиці Фон Контур Списки CSS Просунутий

Шрифти та текст

body{
  font-family: Inter, system-ui, -apple-system, Segoe UI, Roboto, Arial, sans-serif;
  color: var(--text);
}

.footer__text{
  font-size: 14px;
  color: rgba(255,255,255,0.80);
}

Фон і кольори (палітра через CSS-змінні)

:root{
  --teal: #00595C;
  --teal-dark: #032F30;
  --border: #D9D9D9;
  --pill: #F5F5F5;
  --text: #1E1E1E;
  --white: #ffffff;
}

.header{ background: var(--teal); }
.museum{ background: var(--teal-dark); }

Відступи та “контури” (border, radius)

.btn{
  border-radius: 8px;
  padding: 8px 10px;
  border: 1px solid #767676;
}

.museum-vertical{
  border-radius: 14px;
  overflow: hidden;
}

Списки

.about__block ul{
  padding-left: 20px;
}

CSS “просунутий” (анімація + псевдоелементи)

.about-content::before{
  content:"";
  display:block;
  height:80px;
  background-color:#D9FEFF;
  background-image:url("images/ornament.png");
  background-repeat:repeat-x;
  background-size:contain;
  opacity:0.5;
  animation: ornament-drift 40s linear infinite;
}

@keyframes ornament-drift{
  from{ background-position-x: 0; }
  to{ background-position-x: 1000px; }
}

Таблиці (для звіту / приклад)

table{
  width: 100%;
  border-collapse: collapse;
}

th, td{
  border: 1px solid #D9D9D9;
  padding: 8px 10px;
}
